# Netflix Clone Frontend

## Description
Developed a responsive Netflix-style frontend using HTML, CSS, and JavaScript, focusing on clean UI design and smooth user experience.

## Features
- Responsive design  
- Movie rows (horizontal scroll)  
- Hover animations  
- Netflix-style layout  

## How to Run
1. Download the project  
2. Open `index.html` in any web browser  

## Technologies Used
- HTML  
- CSS  
- JavaScript
